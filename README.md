# Universal Website Scraper (Lyftr AI Full-stack Assignment)

This is an MVP "universal website scraper" with a minimal JSON viewer frontend.

## Tech Stack

- Language: Python 3.10+
- Backend: FastAPI
- HTTP client: `httpx`
- HTML parsing: `BeautifulSoup4` + `lxml`
- JS rendering & interactions: Playwright (Python)
- Frontend: Jinja2 template served by FastAPI
- Server runtime: `uvicorn`

## How to Run

```bash
chmod +x run.sh
./run.sh
